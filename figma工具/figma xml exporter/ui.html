Export assets (images)
  • id, name, type, visible, locked
  • x/y, rootX/Y, blendMode, opacity
  • isMask, maskType, absoluteRenderBounds
  • fills, strokes (all paint types)
  • IMAGE fills: imageFile + PNG in ZIP
  • strokeCap, strokeJoin, strokeMiterLimit
  • strokeTopWeight / Right / Bottom / Left
  • effects (with blendMode)
  • cornerRadius / rectangleCornerRadii
  • Auto layout: wrap, counterAxisAlignContent
  • layoutGrids (columns / rows / grid)
  • TEXT: letterSpacing, lineHeight, paragraphSpacing
  • TEXT: textDecoration, textCase, textAutoResize
  • TEXT: maxLines, paragraphIndent, listSpacing
  • Segments: all style fields per run
  • COMPONENT: propertyDefinitions, variantProperties
  • INSTANCE: componentProperties, mainComponentId
  • 9-slice: nineSlice, borders (L/R/T/B), fills[0].imageFile
  • 9-slice PNG: {name}.png — shared across all instances
  • INSTANCE inherits nineSlice from component
  • Tile children: nineSlicePart=true
imageFile filenames are always written in the XML/JSON (Unity-compatible). With assets on: downloads a ZIP with PNG files. With assets off: XML/JSON only.
