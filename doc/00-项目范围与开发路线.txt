Figma ↔ Unity UI 复刻 — 项目范围与开发路线
============================================

更新：2026-06-29
当前阶段：Phase 1 — Figma → Unity（验收通过，可进入 Phase 2 或按需迭代）

一、项目目标（仅此两项）
------------------------

  1. Figma → Unity：Figma 导出后，在 Unity 中复刻 UI（Prefab）     ← Phase 1 已完成
  2. Unity → Figma：Unity 导出后，在 Figma 中复刻 UI（后续）       ← Phase 2

  中间格式：IR（内存 IRNode 树 / 可选 .ir.json 调试落盘）

二、不在范围内
--------------

  Lua / xLua / binding / Button 事件
  isDirty / IRDiffPatcher / 增量更新（v1 全量重建 Prefab）
  ScrollRect / Mask 100% 一致
